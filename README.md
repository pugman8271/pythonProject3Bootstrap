# 21.2 Основы верстки

## Описание:

Разработано простейшее веб-приложение с минимальным функционалом, в дальнейшем будут добавляться новые функции 



## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/pugman8271/pythonProject3Bootstrap.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Тестирование проекта:
```
pytest .\tests\  
```
## Использование:

1. Откройте pycharm.
2. В pycharm откройте проект, указав директорию, куда был скачан проект.
3. С помощью консоли используйте функционал проекта.
